RPG-Konsolenspiel
Beschreibung

Dies ist ein Konsolen-basiertes RPG-Spiel in Python. Der Spieler durchläuft Level, kämpft gegen verschiedene Gegner, sammelt Gold und Erfahrung, kauft Gegenstände im Shop und öffnet Glückskisten. Das Spiel enthält auch eine Storyline, die den Spieler durch verschiedene Kapitel führt.

Features

Levelsystem mit über 100 Leveln und Bossen

Kampfsystem mit HP, Schaden und Spezialfähigkeiten

Shop-System mit Waffen, Rüstungen, Ringen und Haustieren

Glückskistensystem (Cases) mit zufälligen Belohnungen

Story-Modul mit Kapitelprogression

Konsolenbasierte Menüführung

Installation

Python 3.10 oder höher installieren
